It's time to go mobile!

The page looks pretty, aren't it? Ok, at least, it's not broken. But wait, what about the same page on the mobile devices?

At first learn how to simulate a mobile device simulation of the mobile with Chrome Dev Tools

Answer the questions: how to toggle device toolbar, how to switch between different devices, how to create a new device, what is a user agent and how to change it?

1) Find all bugs associated with different devices screens. Go to browser statistic and check the most popular resolutions in the USA. Then, adapt Chrome to each this resolution, find all bugs, document them to the Google Docs.

"F12" key --> 'Toggle device toolbar" (CTRL + SHIFT + M)

2) Our client has a new phone, it's called Xiaomi Mi A2. Let's test if this web page would be pretty on his phone. There are no such phone in Responsive drop-down. Create a new device!

"F12" key --> 'Toggle device toolbar" (CTRL + SHIFT + M) --> Devices drop-down --> "Edit" --> "Add custom device"

Pay attention to selecting a proper 'User agent' - it's very important for correct device simulation!

Similar, you can change a user agent from the "Network condition" tab. Don't forget to reload the page after you change a user agent.

"F12" key --> ⋮ --> More tools --> Network conditions --> User agent

You've learnt how to simulate a screen resolution.
But Chrome can even more! We can check our website on the real Android device. Follow this instruction to open this web page on the real Android device - remote Debugging on Android Devices.
Connect a real android device and inspect this page on it.
"F12" key --> ⋮ --> More tools --> Remote device --> Connected --> Inspect "Chrome dev tools..." page
Now you're ready to inspect this web page on the real Android device, show it your mentor.

Here should be a dark code if you used an Android device or simulated it correctly.

If you tried to simulate Android device with Android user agent or to open a site on the real Android, you could see a dark code. You need this to move to the next step.
