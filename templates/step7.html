1) Open the Console tab. Read about Console API.
Now you should learn how to listen to the disturbance in the power! Sometimes, when you record a video, you need to show to the developer, that you click an item. But how can you print 'click' events in the Console? monitorEvents() function can help you! Write a command into the Console to start monitoring 'click' mouse events.

If monitorEvents() is not enough for you - you can load your own JavaScript to the Console. For example, paste this code:

window.addEventListener("click",function(e) { console.log(e.type, e.target); });

Then click on any UI element. You should see a clicked item in the Console.
